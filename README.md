# creole-nom
WikiCreole markup language parser by nom

### WikiCreole 1.0 markup support list
 - [x] italic
 - [x] bold
 - [x] bullet list
 - [x] numbered list
 - [x] link
 - [x] link with name
 - [x] heading
 - [x] linebreak
 - [x] force linebreak
 - [x] horizontal line
 - [x] image
 - [x] image with title
 - [x] table
 - [x] escape (don't format)

### Extended support
 - [x] link/image/bold/italic in heading/list/table
